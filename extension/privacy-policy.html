JobTrack Privacy Policy

Last updated: April 2026
✅ We do not sell or share your data

What JobTrack Does

JobTrack is a Chrome extension and web dashboard that helps you log job applications with a keyboard shortcut. It saves application data to your personal dashboard powered by Supabase and optionally analyzes job descriptions using AI.

Data We Collect

JobTrack collects the following data when you log a job application:

  • Job title, company name, and job URL (scraped from the current page)
  • Application date and status
  • Notes you optionally add
  • AI-generated role brief, years of experience estimate, and top skills (derived from the job description text)
  • Your email address (for account authentication)

Where Your Data Goes

  • Your Supabase dashboard: All application data is stored in a secure Supabase database, accessible only to your authenticated account via Row Level Security.
  • Your device: Application data is cached locally in chrome.storage.local to power the extension popup. This data never leaves your device except to sync with your dashboard.
  • AI analysis proxy: Job description text is sent to our Cloudflare Worker proxy, which forwards it to Groq's API to generate a role brief, YOE estimate, and skills. No personally identifiable information is included. The text is not stored by our proxy.

Authentication

JobTrack uses Supabase Auth for authentication. You can sign in with email/password or Google OAuth. Your password is never stored by JobTrack — it is handled entirely by Supabase's authentication system. The extension connects to your account via a unique token, which is stored locally in the extension.

Chrome Extension Permissions

The JobTrack extension requests the following permissions:

  • activeTab — to access the current tab when you trigger the keyboard shortcut
  • scripting — to inject the content script for job data extraction
  • storage — to cache application data locally for the popup UI
  • notifications — to show confirmation notifications after logging
  • alarms — to periodically check for stale applications

The content script runs on all URLs so it can extract job details from any job board. It only activates when you press the keyboard shortcut (Cmd+Shift+X / Ctrl+Shift+X) or open the extension popup.

Host Permissions

The extension requests host permission for *.workers.dev to communicate with the AI analysis proxy. The <all_urls> content script permission is required to scrape job details from any job board website — we support 30+ platforms including LinkedIn, Indeed, Greenhouse, Lever, Workday, and many more.

Data Retention

Local storage can be cleared at any time by removing the extension. Data in your Supabase dashboard can be deleted directly from the dashboard. You can export all your data as CSV at any time.

Third-Party Services

  • Supabase — database and authentication. See Supabase's Privacy Policy
  • Groq API — job description text is processed to extract role insights. See Groq's Privacy Policy
  • Cloudflare Workers — our proxy layer. See Cloudflare's Privacy Policy

Single Purpose Description

JobTrack has a single purpose: to help users log and track their job applications. The extension scrapes job details from the current page, enriches them with AI analysis, and saves them to the user's personal dashboard. All features serve this single purpose.

Children's Privacy

JobTrack is not directed at children under 13 and does not knowingly collect data from children.

Changes to This Policy

We may update this policy as the extension evolves. Continued use of JobTrack constitutes acceptance of the updated policy.
